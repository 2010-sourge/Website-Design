# CodingLab — Footer

Projeto de prática de front-end focado na criação de um **footer** (rodapé) responsivo e estilizado, desenvolvido com HTML e CSS.

## 📋 Sobre o projeto

Este projeto consiste em uma página simples que demonstra a construção de um rodapé completo para um site institucional/corporativo, contendo identidade visual, links de navegação organizados por categoria e ícones de redes sociais.

## 🖼️ Preview

O rodapé exibido na página possui:

- **Logo/Nome da marca:** Júlia
- **Ícones de redes sociais:** Facebook, Twitter, Instagram, LinkedIn e YouTube
- **Colunas de links organizadas em:**
  - **Company:** Home, Contact us, About us
  - **Service:** App design, Web Design, Logo Design
  - **Account:** Profile, My account, Preferences

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- Font Awesome (ou similar) para os ícones de redes sociais

## 🚀 Como executar

1. Clone ou baixe este repositório.
2. Abra o arquivo `index.html` diretamente no navegador, **ou**
3. Utilize uma extensão como o **Live Server** (VS Code) para rodar em ambiente local, por exemplo:
   ```
   http://127.0.0.1:5500/index.html
   ```
